Theres a several Wi-fi managers available, both cli & gui, depending on your DEs or workflow preference.

  > INFO: [ CLI: ]
  ==> iwd (internet wirless daemon)
    * minimal and fast wi-fi management tool by intel
    
    _ bashcmd: yay -S iwd
    _ bashcmd: 
      + systemctl start iwd   # start service
      + systemctl enable iwd  # enable service

    - bashcmd:
      * use it with iwctl
      + iwctl           
      + device list
      + station <device> scan
      + station <device> get-networks
      + station <device> connect <SSID>

  ==> nmcli (network manager cli) 
    * widely used networking utility with a commmand-line tool nmcli

    - bashcmd: yay -S network manager
    _ bashcmd: 
      + systemctl start NetworkManager # start service
      + systemctl enable NetworkManager # enable service
    - nmcli device wifi list
    - nmcli device wifi connect "<ssid>" password "<password"

  ==> wpa_supplicant 
    * A lightweight tool for connecting to wifi
    * requires manula configuration of /etc/wpa_supplicant/

  > INFO: [ GUI: ] 
  ==> NetworkManager Applet   
    * Comes with GUI frontends

    bashcmd: pacman -S network-manager network-manager-applet

    bashcmd:
      sudo systemctl enable NetworkManager
      sudo systemctl start NetworkManager

    bashcmd: invoke 
      - nm-applet
